Andamio is a protocol for decentralized treasury management and contribution tracking that enables transparent project coordination on the Cardano blockchain. It provides tools for contributor onboarding, on-chain credentialing, permissionless reputation, and accountable fund allocation that empower teams to collaborate with high levels of transparency and trust.

These docs are a comprehsive guide to Andamio Protocol, including:

- Andamio Protocol: How it works
- Andamio Tokenomics: How Andamio Network Tokens support the protocol
- Andamio APIs: How developers can build on top of Andamio
- Andamio Reference Implementation: How the Andamio Platform at app.andamio.io works
