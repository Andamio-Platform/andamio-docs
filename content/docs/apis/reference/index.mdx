# Andamio Express API Reference

This section provides a complete reference for the Andamio Express API. The documentation is automatically generated from our OpenAPI specification.

## Overview

The Andamio API allows you to interact with various components of the Andamio platform, including:

- Network components (Aggregate, Global State, Validators)
- Course management
- Project management
- Transaction operations

## Available Servers

The API is available on the following servers:

- Development: `http://localhost:3000`
- Testing: `https://dev.andamio.io`
- Pre-production: `https://indexer-preprod-507341199760.us-central1.run.app`
- Production: `https://mainnet-andamio-express-api-507341199760.us-central1.run.app`

## Authentication

Most endpoints require authentication via an Access Token Alias parameter.
